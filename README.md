## 🚀 Featured Projects

### 🛡️ AI-Powered UPI Fraud Detection System
**Real-Time ML Risk Scoring • Behaviour Analysis • Fraud Dashboard**

A complete fraud detection system inspired by real banking systems. It analyzes transaction patterns like timing, frequency, device changes, and user behavior to classify transactions as **ALLOW, FLAG, or BLOCK**.

#### 🔹 Tech Stack
- **Backend:** Python, Flask, SQLite  
- **Frontend:** HTML, Bootstrap, Jinja2, Chart.js  
- **ML:** RandomForestClassifier, Pandas, NumPy, Scikit-learn  

#### 🔹 Features
- Real-time fraud prediction  
- Behavioral risk scoring  
- API-based prediction system  
- Dashboard with analytics  
- Transaction logging
